IDP: Resources

Platform

In this project, we will be used an Arduino (or OrangePip) as the micro-controller. The Arduino is a low-cost microcontroller which is programmed in C++ via USB and has digital inputs/outputs, analogue inputs and PWM sockets. This allows for different sensors/output devices to be controlled. The mechanical system will be developed using laser cutting, 3D printing and other rapid fabrication methods.
